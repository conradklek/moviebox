# MovieBox

A movie-browsing app built with Next.js and [hosted on Vercel](https://moviebox-six-eta.vercel.app/)

## Installation

```bash
git clone git@github.com:conradklek/moviebox.git

cd moviebox

npm install

npm run dev
```
